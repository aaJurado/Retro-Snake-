# Retro Snake 

A retro-style Snake  game built with Python and Pygame.

## Description

Retro Snake  is a desktop game where the player controls a snake, eats food, grows longer, and avoids crashing into the walls or itself. The game includes difficulty modes, score tracking, pause/resume functionality, and persistent high-score storage.

## Features

- Real-time snake movement
- Keyboard controls using Arrow Keys or WASD
- Easy, Medium, and Hard difficulty modes
- Score tracking
- High score saved to a local file
- Pause and resume feature
- Collision detection for walls and snake body
- Retro arcade-style visuals

## Technologies Used

- Python
- Pygame
- File handling
- Lists and tuples
- Functions
- Collision detection
- Game loop logic
